A Galeria de Ótica de Raios

Esta é uma coleção de simulações feitas utilizando a app web, que podem ser utilizadas como demonstrações interativas de ótica. Contribuições de novos elementos são bem-vindas. Veja O guia de contribuição para mais informação. Para reportar um erro, por favor abra um issue no GitHub ou mande um email para ray-optics@phydemo.app.
